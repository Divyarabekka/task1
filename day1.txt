3.Read about IP address, port, HTTP methods, MAC address?

IP ADDRESS: it assigned by an internet service provider (ISP).IP addresses are the identifier that  allows  information to be sent between devices on a network:	
PORT: is unique identify number in a  specific service connection.
http methods:
      get : read data
      post : insert data
      put : update or modify data
      delete :delete data.

MAC ADDRESS: A MAC address (media access control address) is a 12-digit hexadecimal number
               assigned to each device connected to the network. Primarily specified as a unique identifier 
               during device manufacturing.
